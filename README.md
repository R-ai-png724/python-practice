

# Python Basic Programming

This repository contains basic Python programs written while learning Python.
It is mainly for practice and understanding core concepts of Python programming.

---

## What this repository includes

- Basic syntax of Python
- Variables and data types
- Input and output programs
- Conditional statements (if-else)
- Loops (for, while)
- Functions
- Lists, tuples, sets and dictionaries
- Simple programs like Fibonacci, factorial, prime number, etc.

---

## Why I made this repository

I created this repository to:
- Practice Python regularly
- Improve problem-solving skills
- Keep my learning organised
- Help other beginners who are starting Python
